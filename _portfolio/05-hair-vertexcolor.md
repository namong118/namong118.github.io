`3DsMax` `MaxScript` `Unreal`

헤어 지오메트리의 UV를 자동으로 정렬하고, UV 정보 기반으로 Gradient Texture → Vertex Color를 자동으로 베이크하기 위한 스크립트입니다.

기존 헤어 제작 과정(ID 분리 → UV 배열 → 그라디언트 텍스처 적용 → Vertex Color 베이크)의 반복 작업을 자동화할 목적으로 제작했습니다.
