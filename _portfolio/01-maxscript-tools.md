3ds Max 작업 파이프라인 자동화를 목적으로 제작한 MaxScript 툴들입니다.

---

### Distance Measurement Tool
`3DsMax` `MaxScript`

선택한 버텍스나 오브젝트 간의 거리를 빠르게 계산해주는 툴입니다. 버튼 클릭 한 번으로 거리 값을 계산하여 UI에 실시간으로 표시합니다.

---

### Auto SnapMesh Tool
`3DsMax` `MaxScript` `Unreal`

선택한 오브젝트를 일정 프레임 간격으로 자동 스냅샷(Snapshot)하여 각 프레임의 형태를 Editable Poly로 변환 및 병합하는 툴입니다. 애니메이션된 오브젝트의 변형 상태를 정적인 메쉬로 추출하거나 시뮬레이션 결과를 Bake할 때 유용합니다.

---

### UV Controller
`3DsMax` `MaxScript`

Unwrap UVW Modifier 환경에서 선택된 UV를 간편하게 이동(Move) 및 스케일 조정(Scale)할 수 있도록 돕는 UV 편집 자동화 툴입니다.

---

### Pivot Controller
`3DsMax` `MaxScript`

오브젝트의 Pivot을 손쉽게 제어하고, Vertex / Edge / Face 기준으로 pivot 객체(Point / Bone / 선택 오브젝트)를 생성할 수 있는 툴입니다.

---

### Hair VertexColor Tool
`3DsMax` `MaxScript` `Unreal`

헤어 지오메트리의 UV를 자동으로 정렬하고, UV 정보 기반으로 Gradient Texture → Vertex Color를 자동으로 베이크하기 위한 스크립트입니다.

기존 헤어 제작 과정(ID 분리 → UV 배열 → 그라디언트 텍스처 적용 → Vertex Color 베이크)의 반복 작업을 자동화할 목적으로 제작했습니다.
